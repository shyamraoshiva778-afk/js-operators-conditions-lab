# Learning Path

## Week 1: Operators & Basics

### Day 1-2: Arithmetic & Comparison
- Start with: Calculator project
- Learn: +, -, *, /, %
- Practice: Even/Odd checker

### Day 3-4: Logical Operators
- Practice: Largest Number (&&)
- Practice: Login System (&&)

### Day 5-7: Conditional Basics
- if/else: Calculator, Even/Odd
- if/else if: Grade System, Number Type
- Ternary: Voting Check
- Switch: Menu Calculator

## Projects Progression

1. **Calculator** - Basic operations
2. **Even/Odd** - First condition
3. **Number Type** - if/else if ladder
4. **Largest Number** - Multiple conditions
5. **Voting Check** - Ternary operator
6. **Grade System** - Range checking
7. **Login System** - Multiple checks
8. **Discount** - Tiered conditions
9. **Electricity Bill** - Slab system
10. **Menu** - Switch statement
11. **Leap Year** - Complex logic
12. **Student Result** - Complete system

## Tips for Learning

1. **Understand first, then code** - Read the concept before starting
2. **Practice daily** - Do at least 1 project per day
3. **Experiment** - Modify code to see what happens
4. **Debug** - Don't fear errors, learn from them
5. **Build gradually** - Don't jump to complex projects
